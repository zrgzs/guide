# 4.1.1 - 使用 telnet

telnet 是一种用于网络编程的非常强大的调试工具，可以在命令 shell 中输入 telnet 来启动它。

  • 注：在 Windows 中，需要激活 telnet。要激活它，需要到 “控制面板” ，选择 “程序” ，点击 “打开 / 关闭 Windows 特性”， 然后选择 “Telnet” 客户带复选框。
