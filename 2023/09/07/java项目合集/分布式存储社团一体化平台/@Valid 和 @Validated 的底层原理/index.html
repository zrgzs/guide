# 2 正文

首先我们在 Controller 层的方法参数中添加一个 @Email 的注解用于校验单个参数是否为合法的邮箱格式。然后在 Controller 类上添加 @Validated 注解，让 Springboot 进行 ** 参数校验。** 那么 Sprinboot 是如何完成参数校验的呢？这里就以 @Email 注解为例。

# 2.1 找到注解的处理器

进入 @Email 的源代码之后，使用 Ctrl + 鼠标左键 即可查看该注解的所有用法，在所有用法里可以看见一个类： EmailValidator ，从命名和该类所在的包可以断定： org.hibernate.validator.internal.constraintvalidators.bv.EmailValidator 即为校验被 @Email 所标注的参数的。 于是我们在其 isValid() 中打个断点，并调试启动本项目。
